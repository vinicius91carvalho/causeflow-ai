## Overview

Hosted subscription, plan upgrade, and payment-method APIs are **not** part of the open-source product path.
Self-hosted tenants do not select a commercial plan after organization creation.

**Deploy CauseFlow:** clone the [CauseFlow monorepo](https://github.com/causeflow), run `./init.sh`, and continue with the [Quickstart](/getting-started/quickstart).

## OSS behavior

| Outcome | Description |
| --- | --- |
| Not an OSS product path | No plan upgrades, invoices, or payment methods |
| Self-host instead | Local auth → create organization → dashboard home |

Published docs: [GitHub Pages](https://vinicius91carvalho.github.io/causeflow-ai/).

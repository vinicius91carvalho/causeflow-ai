## Overview

In open-source deployments, usage concepts describe **operational load** (investigations and events), not a commercial billing period or paid quota table.
See [Usage and quotas](/billing/usage-and-credits) for how those counters work on a self-hosted stack.

**Deploy CauseFlow:** clone the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai), run `./init.sh`, and open the local dashboard at `http://localhost:3001`.

## OSS guidance

| Topic | Guidance |
| --- | --- |
| Limits | Bounded by your compute and LLM providers, not a hosted plan |
| Monitoring | Watch Core/worker resource usage and investigation volume |
| Docs | [Open-source usage](/billing/plans) and [GitHub Pages](https://vinicius91carvalho.github.io/causeflow-ai/docs/) |

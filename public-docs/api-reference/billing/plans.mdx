## Overview

Returns the plan catalog exposed by your CauseFlow Core API.
In **open-source / self-hosted** deployments, this is a single **Community** plan with no monthly price — commercial Stripe tiers are not part of the OSS product path.

**Authentication:** Public endpoint — no JWT required.

<Note>
  Mutating billing endpoints (`POST /v1/billing/checkout`, `POST /v1/billing/portal`, credit purchase, and similar) return **410 Gone** in the OSS runtime.
  Deploy CauseFlow with `./init.sh` from the [CauseFlow monorepo](https://github.com/causeflow) instead of purchasing a hosted plan.
</Note>

## Response

Returns `200 OK`.

```json
{
  "plans": [
    {
      "planId": "plan_community",
      "name": "Community",
      "priceMonthlyUsd": 0,
      "features": {
        "investigationsPerMonth": "unlimited",
        "eventsPerMonth": "unlimited",
        "apiRateLimitPerMinute": 2000,
        "customSkills": true,
        "relayAccess": true
      }
    }
  ]
}
```

| Field | Type | Description |
| --- | --- | --- |
| `planId` | string | Unique plan identifier |
| `name` | string | Human-readable plan name |
| `priceMonthlyUsd` | number | Monthly price in USD (`0` for OSS Community) |
| `features.investigationsPerMonth` | integer \| `"unlimited"` | Investigation quota (unmetered in OSS) |
| `features.eventsPerMonth` | integer \| `"unlimited"` | Ingested events quota (unmetered in OSS) |
| `features.apiRateLimitPerMinute` | integer | API rate limit enforced by Core |
| `features.customSkills` | boolean | Custom investigation skills available |
| `features.relayAccess` | boolean | Relay deployment included |

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `429` | `rate_limit_exceeded` | Too many requests |

## Examples

<CodeGroup>

```bash curl
curl http://localhost:3099/v1/billing/plans
```

```typescript TypeScript
const response = await fetch("http://localhost:3099/v1/billing/plans");
const { plans } = await response.json();

plans.forEach((p: { name: string; priceMonthlyUsd: number }) =>
  console.log(`${p.name}: ${p.priceMonthlyUsd === 0 ? "free (OSS)" : `$${p.priceMonthlyUsd}/mo`}`)
);
```

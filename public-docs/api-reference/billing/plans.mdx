## Overview

In **open-source / self-hosted** deployments, CauseFlow does not expose a paid plan catalog.
Commercial subscription and payment APIs are removed from the OSS runtime — use free self-host setup instead of purchasing a hosted plan.

**Deploy CauseFlow:** run `./init.sh` from the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai), then follow [Open-source usage](/billing/plans) and the [published docs](https://vinicius91carvalho.github.io/causeflow-ai/docs/).

## OSS behavior

| Outcome | Description |
| --- | --- |
| Not an OSS product path | Paid plan lists, hosted upgrades, and payment flows are out of scope |
| Self-host instead | Clone the monorepo, run `./init.sh`, open the local dashboard |

## Examples

Operators should call Core health and product APIs on the local stack, not a hosted billing catalog:

```bash
curl http://localhost:3099/health
```

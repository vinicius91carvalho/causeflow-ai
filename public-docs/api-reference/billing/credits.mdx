## Overview

Hosted credit balances and quota packs are **not** part of the open-source product path.
Self-hosted CauseFlow does not sell or meter investigation credits against a commercial plan.

**Deploy CauseFlow:** clone the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai), run `./init.sh`, and follow [Open-source usage](/billing/plans).

## OSS behavior

| Outcome | Description |
| --- | --- |
| Not an OSS product path | No credit purchase or plan-quota balance |
| Self-host instead | Monitor load via your infra and [Usage and quotas](/billing/usage-and-credits) |

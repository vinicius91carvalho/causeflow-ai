## Overview

Credit purchase and payment-method flows are **not** available in the open-source self-host distribution.
Capacity is bounded by the compute and LLM providers you configure — not by purchasing hosted quota packs.

**Deploy CauseFlow:** clone the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai), run `./init.sh`, and follow [Open-source usage](/billing/plans).

## OSS behavior

| Outcome | Description |
| --- | --- |
| Not an OSS product path | No payment methods, credit packs, or hosted checkout |
| Self-host instead | Use local Core + dashboard; tune your own infra limits |

See [Usage and quotas](/billing/usage-and-credits) for how investigations and events are counted on self-hosted stacks.

---

## Overview

Triggers the AI triage pipeline for an incident. The triage agent analyzes the incident title, description, and any available signals to classify severity, produce a summary, and recommend which specialist agents should investigate.

**Required role:** `admin` or `member`

## Path parameters

<ParamField path="incidentId" type="string" required>
  The unique identifier of the incident to classify. Example: `inc_01HX9VTPQR3KF8MZWBYD5N6JCE`.
</ParamField>

## Response

Returns `200 OK` with the triage classification result.

```json
{
  "incidentId": "inc_01HX9VTPQR3KF8MZWBYD5N6JCE",
  "priority": "critical",
  "summary": "A latency spike on prod-db-01 is likely caused by a slow query introduced in a recent deployment. Immediate database and performance analysis is recommended.",
  "confidence": 0.91,
  "suggestedAgents": [
    "database-specialist",
    "performance-analyzer",
    "deployment-tracker"
  ]
}
```

| Field             | Type     | Description                                                                       |
| ----------------- | -------- | --------------------------------------------------------------------------------- |
| `incidentId`      | string   | The incident that was classified                                                   |
| `priority`        | string   | AI-determined priority: `critical`, `high`, `medium`, or `low`                   |
| `summary`         | string   | Brief human-readable summary of the incident and recommended next steps           |
| `confidence`      | number   | Model confidence score from `0.0` to `1.0`. Values above `0.85` are high confidence. |
| `suggestedAgents` | string[] | List of specialist agent IDs recommended for the investigation phase              |

<Note>
  To view the full triage evidence including model details and token usage, use [GET /v1/triage/{incidentId}/evidence](/api-reference/triage/get-evidence).
</Note>

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/triage/inc_01HX9VTPQR3KF8MZWBYD5N6JCE \
  -X POST \
  -H "Authorization: Bearer <token>"
```

```typescript TypeScript
const incidentId = "inc_01HX9VTPQR3KF8MZWBYD5N6JCE";

const response = await fetch(
  `https://api.causeflow.ai/v1/triage/${incidentId}`,
  {
    method: "POST",
    headers: {
      Authorization: `Bearer ${token}`,
      "Content-Type": "application/json",
    },
  }
);

const triage = await response.json();

console.log(`Priority: ${triage.priority} (confidence: ${triage.confidence})`);
console.log(`Suggested agents: ${triage.suggestedAgents.join(", ")}`);
```

</CodeGroup>

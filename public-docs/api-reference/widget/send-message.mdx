## Overview

<Warning>
  **Roadmap — not yet available.** The embeddable widget is a future feature.
</Warning>

Sends a message from the user to the widget AI assistant. The assistant's response is delivered via the [widget SSE stream](/api-reference/widget/stream) — this endpoint accepts the message and returns immediately.

**Authentication:** Widget session token (pass as `Authorization: Bearer <sessionToken>` or `X-Widget-Token` header).

## Request headers

<ParamField header="Authorization" type="string" required>
  Widget session token. Format: `Bearer <sessionToken>`. Obtain from [Create widget session](/api-reference/widget/create-session).
</ParamField>

## Path parameters

<ParamField path="sessionId" type="string" required>
  The widget session identifier from [Create widget session](/api-reference/widget/create-session).
</ParamField>

## Request body

<ParamField body="message" type="string" required>
  The user's message. Maximum 2,000 characters.
</ParamField>

<ParamField body="attachmentIds" type="array">
  Optional list of previously uploaded attachment IDs to include with this message.
</ParamField>

## Response

Returns `202 Accepted`. The assistant's reply streams on `GET /v1/widget/sessions/{sessionId}/stream`.

```json
{
  "messageId": "wmsg_01HX9VTPQR3KF8MZWBYD5N6JCE",
  "sessionId": "wsess_01HX9VTPQR3KF8MZWBYD5N6JCE",
  "status": "processing",
  "submittedAt": "2024-04-01T14:30:00Z"
}
```

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `400` | `validation_error` | `message` is empty or exceeds length limit |
| `401` | `unauthorized` | Missing or invalid session token |
| `404` | `session_not_found` | Session does not exist or has expired |
| `429` | `rate_limit_exceeded` | Too many messages in this session |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/widget/sessions/wsess_01HX9VTPQR3KF8MZWBYD5N6JCE/messages \
  -X POST \
  -H "Authorization: Bearer wst_live_EXAMPLE_TOKEN_REDACTED" \
  -H "Content-Type: application/json" \
  -d '{ "message": "What is the root cause of inc_EXAMPLE_01JX?" }'
```

```typescript TypeScript
const response = await fetch(
  "https://api.causeflow.ai/v1/widget/sessions/wsess_01HX9VTPQR3KF8MZWBYD5N6JCE/messages",
  {
    method: "POST",
    headers: {
      Authorization: `Bearer ${sessionToken}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ message: "What is the root cause of inc_EXAMPLE_01JX?" }),
  }
);

const { messageId } = await response.json();
console.log("Message sent:", messageId, "— listening on stream for reply");
```

---
title: "Close widget session"
api: "DELETE /v1/widget/sessions/{sessionId}"
description: "Terminate an active widget session and invalidate the session token."
---

## Overview

<Warning>
  **Roadmap — not yet available.** The embeddable widget is a future feature.
</Warning>

Closes a widget session immediately, invalidating the session token and terminating any active SSE streams. Use this when the user ends a chat session or navigates away.

**Authentication:** Widget session token.

## Request headers

<ParamField header="Authorization" type="string" required>
  Widget session token. Format: `Bearer <sessionToken>`. Obtain from [Create widget session](/api-reference/widget/create-session).
</ParamField>

## Path parameters

<ParamField path="sessionId" type="string" required>
  The widget session identifier to close.
</ParamField>

## Response

Returns `204 No Content` on success.

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `401` | `unauthorized` | Missing or invalid session token |
| `404` | `session_not_found` | Session does not exist or already closed |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/widget/sessions/wsess_01HX9VTPQR3KF8MZWBYD5N6JCE \
  -X DELETE \
  -H "Authorization: Bearer wst_live_EXAMPLE_TOKEN_REDACTED"
```

```typescript TypeScript
const response = await fetch(
  "https://api.causeflow.ai/v1/widget/sessions/wsess_01HX9VTPQR3KF8MZWBYD5N6JCE",
  {
    method: "DELETE",
    headers: { Authorization: `Bearer ${sessionToken}` },

if (response.status === 204) {
  console.log("Widget session closed");
}
```

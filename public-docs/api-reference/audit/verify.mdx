---
title: "Verify audit integrity"
api: "POST /v1/audit/verify"
description: "Verify the tamper-evidence hash of one or more audit log entries."
---

## Overview

Verifies the SHA-256 integrity hash of audit entries. Use this during compliance audits or incident reviews to confirm that audit records have not been altered since they were written.

**Required role:** `admin`

## Request headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token. Format: `Bearer <your-jwt>`.
</ParamField>

## Request body

<ParamField body="auditIds" type="array" required>
  List of audit entry IDs to verify. Maximum 100 per request.
</ParamField>

## Response

Returns `200 OK`.

```json
{
  "results": [
    {
      "auditId": "aud_01HX9VTPQR3KF8MZWBYD5N6JCE",
      "verified": true,
      "computedHash": "sha256:a4b2c3d4e5f6...",
      "storedHash": "sha256:a4b2c3d4e5f6..."
    }
  ],
  "allVerified": true
}
```

| Field | Type | Description |
| --- | --- | --- |
| `results[].auditId` | string | The audit entry ID |
| `results[].verified` | boolean | `true` if computed and stored hashes match |
| `results[].computedHash` | string | Hash recomputed from the entry data |
| `results[].storedHash` | string | Hash stored at write time |
| `allVerified` | boolean | `true` if every entry in the request verified successfully |

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `400` | `validation_error` | `auditIds` is empty or exceeds 100 entries |
| `401` | `unauthorized` | Missing or invalid JWT |
| `403` | `forbidden` | Caller lacks `admin` role |
| `404` | `audit_entry_not_found` | One or more audit IDs do not exist |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/audit/verify \
  -X POST \
  -H "Authorization: Bearer eyJhbGc..." \
  -H "Content-Type: application/json" \
  -d '{ "auditIds": ["aud_01HX9VTPQR3KF8MZWBYD5N6JCE"] }'
```

```typescript TypeScript
const response = await fetch("https://api.causeflow.ai/v1/audit/verify", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    auditIds: ["aud_01HX9VTPQR3KF8MZWBYD5N6JCE"],
  }),
});

const { results, allVerified } = await response.json();
if (!allVerified) {
  const tampered = results.filter((r: { verified: boolean }) => !r.verified);
  console.error("INTEGRITY FAILURE — tampered entries:", tampered.map((r: { auditId: string }) => r.auditId));
} else {
  console.log("All audit entries verified");
}
```

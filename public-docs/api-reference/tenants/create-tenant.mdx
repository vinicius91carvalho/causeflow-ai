## Overview

Creates a new tenant (organization) in CauseFlow. Each tenant is an isolated workspace with its own users, incidents, and configuration.

**Required role:** `admin`

## Request body

<ParamField body="name" type="string" required>
  The display name of the tenant. Used in the dashboard and notifications. Example: `"Acme Corp"`.
</ParamField>

<ParamField body="slug" type="string" required>
  A URL-safe identifier for the tenant. Must be lowercase alphanumeric characters and hyphens only. Example: `"acme-corp"`. Must be globally unique — returns `409 Conflict` if already taken.
</ParamField>

<ParamField body="ownerEmail" type="string" required>
  Email address of the tenant owner. Must be a valid email address. The owner receives an invitation to activate their account.
</ParamField>

<ParamField body="plan" type="string">
  Optional deployment label for the tenant. In open-source self-host this is informational only (there is no paid plan selection). Defaults to `community` when omitted.
</ParamField>

<ParamField body="settings" type="object">
  Optional AWS integration settings for the tenant.

  <Expandable title="settings properties">
    <ParamField body="settings.awsRoleArn" type="string">
      The ARN of the AWS IAM role CauseFlow will assume for AWS integrations. Example: `"<your-aws-iam-role-arn>"`.
    </ParamField>

    <ParamField body="settings.awsExternalId" type="string">
      The external ID used when assuming the AWS IAM role. Used as an additional trust condition on the role's trust policy.
    </ParamField>

    <ParamField body="settings.awsRegion" type="string">
      The primary AWS region for this tenant's infrastructure. Example: `"us-east-1"`.
    </ParamField>
  </Expandable>
</ParamField>

## Response

Returns `201 Created` with the newly created tenant.

```json
{
  "tenantId": "ten_EXAMPLE_01HX9VTPQR3KF8MZ",
  "name": "Acme Corp",
  "slug": "acme-corp",
  "status": "active",
  "plan": "community",
  "createdAt": "2026-04-01T12:00:00Z"
}
```

| Field       | Type   | Description                                              |
| ----------- | ------ | -------------------------------------------------------- |
| `tenantId`  | string | Unique identifier for the newly created tenant           |
| `name`      | string | The tenant display name                                  |
| `slug`      | string | The URL-safe tenant identifier                           |
| `status`    | string | Always `"active"` for newly created tenants              |
| `plan`      | string | Deployment label (`community` in OSS; not a paid tier)   |
| `createdAt` | string | ISO 8601 timestamp of when the tenant was created        |

### Error responses

| Status | Error code         | Description                                         |
| ------ | ------------------ | --------------------------------------------------- |
| `400`  | `validation_error` | A required field is missing or fails validation     |
| `409`  | `slug_conflict`    | A tenant with the provided `slug` already exists    |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/tenants \
  -X POST \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp",
    "slug": "acme-corp",
    "ownerEmail": "owner@acme.com",
    "plan": "community",
    "settings": {
      "awsRoleArn": "<your-aws-iam-role-arn>",
      "awsExternalId": "causeflow-ext-id-acme",
      "awsRegion": "us-east-1"
    }
  }'
```

```typescript TypeScript
const response = await fetch("https://api.causeflow.ai/v1/tenants", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Acme Corp",
    slug: "acme-corp",
    ownerEmail: "owner@acme.com",
    plan: "community",
    settings: {
      awsRoleArn: "<your-aws-iam-role-arn>",
      awsExternalId: "causeflow-ext-id-acme",
      awsRegion: "us-east-1",
    },
  }),
});

if (response.status === 409) {
  throw new Error("A tenant with that slug already exists. Choose a different slug.");
}

const tenant = await response.json();
console.log("Created tenant:", tenant.tenantId);
```

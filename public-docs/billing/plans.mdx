CauseFlow is available as **free, open-source software** you run on your own infrastructure.
There is no hosted plan catalog, paid upgrade path, or payment step in this distribution.

## Get started

<CardGroup cols={2}>
  <Card title="Self-host with Docker" icon="docker" href="https://github.com/vinicius91carvalho/causeflow-ai">
    Clone the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai) and run `./init.sh` from the repository root to start the local stack (website, dashboard, Core API, and docs).
  </Card>
  <Card title="Read the docs" icon="book" href="https://vinicius91carvalho.github.io/causeflow-ai/">
    Browse the published documentation on [GitHub Pages](https://vinicius91carvalho.github.io/causeflow-ai/docs/) for quickstart, integrations, API reference, and Relay setup.
  </Card>
</CardGroup>

After `./init.sh` completes, use these local URLs:

| Service | URL |
|---------|-----|
| Dashboard | `http://localhost:3001` |
| Core API | `http://localhost:3099` |
| Docs (local compose) | `http://localhost:5181` |

Sign in through the local dashboard, create your organization, and connect integrations.
No plan selection or payment step is required.

## What's included

Every self-hosted deployment includes the full product surface:

| Capability | Included |
|------------|----------|
| All integrations in the catalog | Yes |
| AI triage and investigations | Yes |
| Knowledge base and memory | Yes |
| Remediation proposals | Yes |
| RBAC and audit trail | Yes |
| CauseFlow Relay (optional) | Yes |

Capacity is bounded by the compute and LLM providers you configure, not by a commercial quota table.

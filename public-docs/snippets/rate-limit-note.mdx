---
title: Rate limit note
description: Reusable note explaining CauseFlow API rate limits, the 429 response, and the X-RateLimit response headers.
---

<Note>
  Rate limits vary by plan. When you exceed your rate limit, the API returns `429 Too Many Requests`. Check the `X-RateLimit-Remaining` and `X-RateLimit-Reset` response headers to manage your request volume.
</Note>

---
title: Authentication header
description: Reusable snippet showing the JWT Authorization header required by all authenticated CauseFlow API endpoints.
---

```bash
Authorization: Bearer <your-jwt-token>
```

All authenticated endpoints require a valid JWT token in the `Authorization` header. Obtain your token by signing in to the local dashboard after `./init.sh` (`http://localhost:3001`), or see the [CauseFlow monorepo](https://github.com/vinicius91carvalho/causeflow-ai).

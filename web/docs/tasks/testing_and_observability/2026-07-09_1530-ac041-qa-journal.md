# QA Journal: WI-AC-041 (Playwright test.beforeEach blocks analytics + tracker domains)

## Date: 2026-07-09

## Result: PASS - implementation=true, qa=true

### Independent Verification

1. **`tests/audit.spec.ts` and `tests/visual-functional.spec.ts` both declare `test.beforeEach` with `page.route()` blocking** - VERIFIED
   - `grep -n 'page.route' tests/audit.spec.ts tests/visual-functional.spec.ts` confirms both files have `page.route('**/*', ...)` with `route.abort()` for blocked patterns

2. **Block list covers the documented domains** - VERIFIED
   - `google-analytics.com` (covers `www.google-analytics.com` and gtag's `collect` endpoint)
   - `clarity.ms` and `www.clarity.ms` (covers `*.clarity.ms` and `clarity.ms` — Clarity's session-replay endpoints)
   - `intercom.io` and `widget.intercom.io` (covers Intercom messenger and API)
   - Additional domains: googletagmanager, analytics.google, facebook, doubleclick, googlesyndication, adservice, segment, sentry
   - `shouldBlock()` uses `url.includes(pattern)` which matches subdomains without extra patterns

3. **Runtime analytics blocking verified** - VERIFIED
   - All 18 audit tests pass (chromium-desktop project) with zero console errors
   - Focused verification test confirmed:
     - `shouldBlock('https://www.google-analytics.com/g/collect')` returns `true`
     - `shouldBlock('https://clarity.ms/collect')` returns `true`
     - `shouldBlock('https://www.clarity.ms/collect')` returns `true`
     - `shouldBlock('https://sub.clarity.ms/collect')` returns `true` (covers `*.clarity.ms`)
     - `shouldBlock('https://widget.intercom.io/widget/abc123')` returns `true`
     - `shouldBlock('https://api.intercom.io/messages')` returns `true`
     - Legitimate app traffic (`causeflow.ai`, `localhost`) correctly returns `false`
   - Runtime page navigation confirms no analytics requests leak through during homepage or product page load

4. **Tests use real chromium browser** - VERIFIED
   - All tests execute against the production build (`next start`) via Playwright's chromium engine
   - No mock browser or simulated HTTP layer used

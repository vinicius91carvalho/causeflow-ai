Currently: Desktop 100%, Mobile 94%. Target: Mobile 100%.

## Issues from PageSpeed Insights

1. **Render-blocking CSS** (450ms savings) — Two CSS files blocking initial render
2. **Legacy JavaScript polyfills** (10 KiB) — Unnecessary polyfills (Array.at, flat, flatMap, Object.fromEntries, hasOwn, String.trimEnd/trimStart)
3. **Critical request chains** (817ms) — CSS chain through navigation
4. **Unused JavaScript** (35 KiB) — chunks/537-*.js has 35.3 KiB unused

## Root Causes Found

- **No browserslist config** → SWC/Next.js injects polyfills for older browsers
- **All 5 themes loaded in entry.css** → 30+ CSS imports regardless of active theme → bloated CSS
- **No `optimizePackageImports`** → lucide-react, @radix-ui not tree-shaken properly
- **Only 2 dynamic imports** on homepage → most components statically bundled
- **No critical CSS inlining** → both CSS files are render-blocking

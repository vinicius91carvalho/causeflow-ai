# Solutions Library

Institutional knowledge base for CauseFlow AI. Every solved problem becomes searchable documentation.

## Structure

- `infrastructure/` — Environment setup, deployment, tooling solutions
- `patterns/` — Reusable code and workflow patterns
- `bugfixes/` — Bug root causes and fixes
- `performance/` — Optimization techniques with benchmarks
- `security/` — Security patterns and configurations

## How to Use

1. **Before solving a problem:** Search this directory for existing solutions
2. **After solving a problem:** Create a new solution doc if the pattern is reusable
3. **Format:** Use YAML frontmatter + Problem/Root Cause/Solution/Prevention sections
4. **Naming:** `YYYY-MM-DD_descriptive-name.md`

## Tags

Search by tags in YAML frontmatter for quick discovery.

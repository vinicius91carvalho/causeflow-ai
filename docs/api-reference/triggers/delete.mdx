---
title: "Delete trigger"
api: "DELETE /v1/triggers/{triggerId}"
description: "Remove an automation trigger permanently."
---

## Overview

Permanently deletes an automation trigger. Fires a `trigger.deleted` event on the EventBus. In-flight actions already started by this trigger are not affected.

**Required role:** `admin`

## Request headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token. Format: `Bearer <your-jwt>`.
</ParamField>

## Path parameters

<ParamField path="triggerId" type="string" required>
  The trigger identifier. Example: `trg_01HX9VTPQR3KF8MZWBYD5N6JCE`.
</ParamField>

## Response

Returns `204 No Content` on success.

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `401` | `unauthorized` | Missing or invalid JWT |
| `403` | `forbidden` | Caller lacks `admin` role |
| `404` | `trigger_not_found` | No trigger with the given ID |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/v1/triggers/trg_01HX9VTPQR3KF8MZWBYD5N6JCE \
  -X DELETE \
  -H "Authorization: Bearer eyJhbGc..."
```

```typescript TypeScript
const response = await fetch(
  "https://api.causeflow.ai/v1/triggers/trg_01HX9VTPQR3KF8MZWBYD5N6JCE",
  {
    method: "DELETE",
    headers: { Authorization: `Bearer ${token}` },

if (response.status === 204) {

```

---
title: "Update skill"
api: "PATCH /api/v1/tenants/{tenantId}/skills/{skillId}"
description: "Update the configuration or enabled state of an existing skill."
---

## Overview

Partially updates a skill. Send only the fields you want to change â€” all other fields remain unchanged.

<Note>
  Skills use a non-standard base path: `/api/v1/tenants/{tenantId}/skills` â€” not `/v1/`. This is a dedicated module with its own routing prefix.
</Note>

**Required role:** `admin`

## Request headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token. Format: `Bearer <your-jwt>`.
</ParamField>

## Path parameters

<ParamField path="tenantId" type="string" required>
  Your tenant identifier. Example: `ten_EXAMPLE_ABC123`.
</ParamField>

<ParamField path="skillId" type="string" required>
  The skill identifier. Example: `skl_01HX9VTPQR3KF8MZWBYD5N6JCE`.
</ParamField>

## Request body

<ParamField body="description" type="string">
  Updated description. Maximum 500 characters.
</ParamField>

<ParamField body="config" type="object">
  Updated execution configuration. Replaces the existing `config` entirely.
</ParamField>

<ParamField body="inputSchema" type="object">
  Updated input schema. Replaces the existing `inputSchema` entirely.
</ParamField>

<ParamField body="enabled" type="boolean">
  Set to `false` to disable the skill without deleting it.
</ParamField>

## Response

Returns `200 OK` with the updated skill.

```json
{
  "skillId": "skl_01HX9VTPQR3KF8MZWBYD5N6JCE",
  "name": "fetch_runbook",
  "enabled": false,
  "updatedAt": "2024-04-01T12:00:00Z"
}
```

### Error responses

| Status | Error code | Description |
| --- | --- | --- |
| `400` | `validation_error` | Invalid field value or schema |
| `401` | `unauthorized` | Missing or invalid JWT |
| `403` | `forbidden` | Caller lacks `admin` role |
| `404` | `skill_not_found` | No skill with the given ID |

## Examples

<CodeGroup>

```bash curl
curl https://api.causeflow.ai/api/v1/tenants/ten_EXAMPLE_ABC123/skills/skl_01HX9VTPQR3KF8MZWBYD5N6JCE \
  -X PATCH \
  -H "Authorization: Bearer eyJhbGc..." \
  -H "Content-Type: application/json" \
  -d '{ "enabled": false }'
```

```typescript TypeScript
const response = await fetch(
  "https://api.causeflow.ai/api/v1/tenants/ten_EXAMPLE_ABC123/skills/skl_01HX9VTPQR3KF8MZWBYD5N6JCE",
```
